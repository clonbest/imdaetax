# 임대택스 관리 시스템

임대사업자를 위한 전문 세무 관리 웹 애플리케이션입니다. 알림톡 한 번 확인으로 세금계산서 발행부터 부가가치세 신고까지 간편하게 처리할 수 있습니다.

## 🎯 프로젝트 개요

이 프로젝트는 임대사업자가 세무 업무를 효율적으로 관리할 수 있도록 돕는 웹 기반 플랫폼입니다. 청천세무회계 스타일의 깔끔하고 모던한 디자인을 적용하여 사용자 친화적인 인터페이스를 제공합니다.

## ✨ 주요 기능

### 완료된 기능

#### 1. 랜딩 페이지 (index.html)
- **서비스 소개**
  - 세금계산서 발행: 알림톡 확인 한번이면 세금계산서 발행 끝
  - 부가가치세 신고: 세무전문가가 작성, 검토한 신고서를 알림톡으로 확인
  - 세무상담: 친절한 세무전문가가 상담
- **임대택스 장점**
  - 임대세무 전문성: 단순 AI가 아닌 임대업 세무 전문가
  - 간편한 세무관리: 알림톡만 확인하면 모든 처리 완료
  - 합리적인 가격: 부가세 신고대리 수수료보다 저렴한 통합관리
- **서비스 이용료**
  - 라이트: 월 1.5만원 (세금계산서 월 3건 이하)
  - 스탠다드: 월 3만원 (세금계산서 월 10건 이하)
  - 플러스: 월 5만원 (세금계산서 월 20건 이하)
  - *부가세 신고는 사업자번호별 1.5만원
- FAQ 섹션
- 카카오톡 채널 연결 (http://pf.kakao.com/_xcIaUn/chat)

#### 2. 인증 시스템
- **로그인** (login.html)
  - 이메일/비밀번호 인증
  - 로그인 상태 유지 옵션
  - 데모 계정 제공
- **회원가입** (register.html)
  - 신규 사용자 등록
  - 비밀번호 확인 검증
  - 이용약관 동의

#### 3. 대시보드 (dashboard.html)
- 통계 카드 (총 세금계산서, 발행 완료, 처리 중, 총 금액)
- 최근 세금계산서 목록 (상위 5개)
- 부가세 신고 현황 (상위 3개)
- 사용자 정보 표시

#### 4. 세금계산서 관리 (invoices.html)
- 세금계산서 발행 내역 전체 조회
- 필터링 기능 (기간, 상태, 거래처명 검색)
- 페이지네이션
- 상세보기 기능

#### 5. 부가세 신고 관리 (vat-returns.html)
- 부가가치세 신고 내역 조회
- 필터링 기능 (년도, 기수, 상태)
- 매출세액, 매입세액, 납부세액 표시
- 신고 안내 정보

#### 6. 프로필 관리 (profile.html)
- 기본 정보 수정 (이름, 연락처, 사업자정보)
- 비밀번호 변경
- 요금제 정보 확인

## 💾 데이터 저장

### localStorage 사용
현재 버전은 브라우저의 localStorage를 사용하여 데이터를 저장합니다:
- `currentUser` - 현재 로그인한 사용자 정보
- `users` - 회원가입한 사용자 목록
- `userDetails` - 사용자 상세 정보
- `sampleInvoices` - 세금계산서 샘플 데이터
- `sampleVATReturns` - 부가세 신고 샘플 데이터

### RESTful Table API (준비됨)
다음 테이블 스키마가 정의되어 있어 API를 통한 데이터 관리가 가능합니다:

#### invoices (세금계산서)
- id, date, client, businessNumber, supplyAmount, tax, status

#### vat_returns (부가세 신고)
- id, period, salesTax, purchaseTax, paymentTax, refundTax, deadline, paymentDeadline, status

#### users (사용자)
- id, name, email, phone, businessNumber, businessName, address
